# Esque FrontEnd 페이지

## 사용한 기술

```
next를 이용하여 SSR (ServerSideRendering)
typeScript를 사용하려 tsx 파일로 생성 (But javaScript 스타일로 코딩)
```

## Getting Started

```
1. npm install 로 패치지 설치
2. npm run dev 로 실행
```

## 서비스(배포) 주소

```
esque.kr
```
